### 参数
#### QsEllise 参数
| 参数名 | 说明 | 类型 | 默认值 |
| --- | ---- | ---- | ----- |
| content | 内容 | string | - |
| line | 超过多少行使用... | number | 3 |
| expanText | 展开操作的文案 | string | '展开' |
| closeText | 收起操作的文案 | string | '收起' |
